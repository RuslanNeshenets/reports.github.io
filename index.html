Тема: СТРУКТУРА HTML-ДОКУМЕНТА. ВИБІР ПРЕДМЕТНОГО ГАЛУЗІ. РОБОТА З ПОСИЛАННЯМИ, ТАБЛИЦЯМИ, ЗОБРАЖЕННЯМИ, ФОРМАМИ в HTML-ДОКУМЕНТІ.

Мета: придбати практичні навички роботи з HTML-документом, таблицями,формами, зображеннями, посиланнями. Створити шаблон звітного HTML-документом для відображення результатів роботи всіх лабораторних робіт.

У процесі виконання даної лабораторної роботи було створено веб-сторінку із вказанням посилання на сторінку з описом предметної галузі, прикладом таблиці ігор, форм для оцінки сторінки та декількох зображень дизайну сайтів цієї предметної галузі.

Було створено звітний html документ у якому надано можливість переходу до результатів цієї та наступних лабораторних робіт. Також присутня можливість перегляду опису предметного середовища, завдання лабораторної роботи, структури документу, коду таблиць, зображень й інших об'єктів, які використовувались у роботі, форм і висновку.

Робота виконувалась у програмних середовищах VS Code та Sublime Text.
